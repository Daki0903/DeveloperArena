JavaScript Guide

JavaScript is a versatile programming language primarily used for web development. It allows developers to create interactive and dynamic web pages.

JavaScript is the programming language of the Web. 
console.log("Hello, World!");
Getting Started
  • Write JavaScript in a <script> tag or in a separate .js file.
  • Use console.log() to print messages for debugging.
<script>
    console.log("Hello from JS!");
</script>
Variables & Data Types
  • Declare variables with let, const, or var.
  • Common types: String, Number, Boolean, Array, Object.
let name = "Alice";
const age = 30;
var isActive = true;
let numbers = [1, 2, 3];
let person = { name: "Bob", age: 25 };
Functions
  • Functions let you reuse code.
function greet(name) {
    return "Hello, " + name + "!");
}
console.log(greet("World"));
DOM Manipulation
  • Change HTML content and styles dynamically.
document.getElementById("demo").innerText = "Changed!";
Event Handling
  • React to user actions like clicks.
document.querySelector("button").addEventListener("click", function() {
    alert("Button clicked!");
});
