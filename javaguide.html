Java Guide

Java is a versatile and widely-used programming language known for its portability, performance, and extensive libraries.

It is commonly used for building enterprise applications, mobile applications (Android), and web applications.

Java's syntax is similar to C++, making it easier for developers familiar with those languages to learn.

Download Java JDK
Getting Started
  • Install the Java JDK and set up your environment variables.
  • Write Java code in a .java file and compile it with javac.
  • Run your program with java.
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
Variables & Data Types
  • Common types: int, double, boolean, String, char.
  • Variables must be declared with a type.
int age = 25;
double price = 19.99;
boolean isJavaFun = true;
String name = "Alice";
char grade = 'A';
Methods
  • Methods are functions inside classes.
public static int add(int a, int b) {
    return a + b;
}
Classes & Objects
  • Java is object-oriented. Everything is part of a class.
public class Dog {
    String name;
    public Dog(String name) {
        this.name = name;
    }
    public void bark() {
        System.out.println(name + " says woof!");
    }
}

// Usage:
Dog myDog = new Dog("Buddy");
myDog.bark();
Control Structures
  • Use if, else, for, while for logic and loops.
for (int i = 0; i < 5; i++) {
    System.out.println(i);
}

if (age > 18) {
    System.out.println("Adult");
} else {
    System.out.println("Minor");
}
Arrays
  • Arrays store multiple values of the same type.
int[] numbers = {1, 2, 3, 4, 5};
System.out.println(numbers[0]); // prints 1
Inheritance
  • Inheritance lets you create new classes based on existing ones.
class Animal {
    void eat() {
        System.out.println("This animal eats food.");
    }
}

class Dog extends Animal {
    void bark() {
        System.out.println("Woof!");
    }
}

// Usage:
Dog d = new Dog();
d.eat();  // Inherited method
d.bark(); // Dog's own method
Exception Handling
  • Handle errors gracefully using try, catch, and finally.
try {
    int result = 10 / 0;
} catch (ArithmeticException e) {
    System.out.println("Cannot divide by zero!");
} finally {
    System.out.println("This always runs.");
}
Interfaces
  • Interfaces define methods that classes must implement.
interface Animal {
    void makeSound();
}

class Cat implements Animal {
    public void makeSound() {
        System.out.println("Meow");
    }
}
Collections
  • Use collections like ArrayList for dynamic arrays.
import java.util.ArrayList;

ArrayList<String> fruits = new ArrayList<>();
fruits.add("Apple");
fruits.add("Banana");
System.out.println(fruits.get(0)); // prints Apple
